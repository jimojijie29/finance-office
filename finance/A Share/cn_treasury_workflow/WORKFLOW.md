# 中国国债收益率数据工作流 (Tushare版)

## 概述

本工作流使用 Tushare `yc_cb` 接口获取中债国债收益率曲线历史数据，支持完整的各期限数据获取和增量更新。

## 核心参数

| 参数名 | 设置值 | 说明 |
|--------|--------|------|
| `ts_code` | `'1001.CB'` | 国债收益率曲线专用代码（必须设置） |
| `curve_type` | `'0'` | 到期收益率（标准国债收益率） |
| `trade_date` | `'YYYYMMDD'` | 特定交易日 |
| `start_date`/`end_date` | `'YYYYMMDD'` | 日期范围 |

## 数据期限

支持以下期限的国债收益率数据：

| 期限 | 列名 | 说明 |
|------|------|------|
| 1年 | `X1年` | 1年期国债收益率 |
| 2年 | `X2年` | 2年期国债收益率 |
| 3年 | `X3年` | 3年期国债收益率 |
| 5年 | `X5年` | 5年期国债收益率 |
| 7年 | `X7年` | 7年期国债收益率 |
| 10年 | `X10年` | 10年期国债收益率（市场关注重点） |
| 20年 | `X20年` | 20年期国债收益率 |
| 30年 | `X30年` | 30年期国债收益率 |

## 使用方法

### 1. 配置 Tushare Token

已配置环境变量 `TUSHARE_TOKEN`，无需额外操作。

如需使用配置文件，编辑 `tushare_config.txt`：
```
你的_token_在这里
```

### 2. 获取数据

**方式一：自动增量更新（推荐）**
```bash
cd finance/cn_treasury_workflow
python fetch_cn_treasury_tushare.py
```

**方式二：获取最近N天数据**
```bash
python fetch_data_simple.py
```

### 3. 数据分析

```bash
python analyze_cn_treasury.py
```

分析内容包括：
- 最新各期限收益率
- 期限利差分析（10年 - 1年）
- 收益率曲线形态判断
- 移动平均线分析
- 数据完整性统计

## 技术特点

1. **核心参数配置**
   - `ts_code='1001.CB'`: 国债收益率曲线专用代码
   - `curve_type='0'`: 到期收益率（市场标准）

2. **数据获取方式**
   - 按日期循环获取，支持增量更新
   - 自动跳过周末和非交易日
   - 遵守API频率限制（每分钟2次）

3. **数据处理**
   - 原始数据透视转换（长格式→宽格式）
   - 自动处理缺失期限数据
   - 合并新旧数据，避免重复

4. **错误处理**
   - API频次限制自动等待
   - 权限错误友好提示
   - 网络异常自动重试

## 注意事项

1. **API限制**
   - 每分钟最多2次调用
   - 需要足够的Tushare积分权限
   - 部分日期可能无数据（节假日等）

2. **数据延迟**
   - 中债数据通常有1个工作日延迟
   - 当日数据可能无法获取

3. **数据范围**
   - 首次运行获取最近3个月数据
   - 后续运行自动增量更新

## 定时更新

可设置定时任务自动更新数据：

```bash
# Windows任务计划程序
# 每天上午9:30运行
python D:\OpenClawData\.openclaw\workspace\finance\cn_treasury_workflow\fetch_cn_treasury_tushare.py
```

## 故障排除

| 问题 | 原因 | 解决 |
|------|------|------|
| "未找到Tushare Token" | 环境变量未设置 | 检查 `TUSHARE_TOKEN` 环境变量 |
| "权限不足或积分不够" | 积分不足 | 访问 tushare.pro 查看积分要求 |
| "频次限制" | API调用太频繁 | 脚本已自动处理，等待后重试 |
| "无数据" | 该日期无数据 | 可能是节假日或非交易日 |
