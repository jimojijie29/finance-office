# 美国国债收益率数据更新工作流

## 概述

这是一个完整的数据更新工作流，整合了从数据获取到可视化的全部流程。只需运行一个脚本，即可自动完成：

1. 从东方财富数据库获取最新数据
2. 处理并合并到现有数据文件
3. 生成更新的可视化 HTML

## 文件结构

```
us_treasury_workflow/
├── update_us_treasury_data.py    # 主更新脚本
├── us_treasury_workflow_v2.R     # R 可视化脚本
├── us_treasury_yields_all_terms.csv  # 数据文件
├── us_treasury_all_terms.html    # 可视化结果
└── README_UPDATE_WORKFLOW.md     # 使用说明
```

## 使用方法

### 1. 进入工作流目录

```bash
cd us_treasury_workflow
```

### 2. 更新数据（推荐）

```bash
python update_us_treasury_data.py
```

脚本会自动：
- 检测现有数据的最后日期
- 获取从最后日期到今天的最新数据
- 合并新旧数据（自动去重）
- 验证数据质量
- 生成更新的可视化 HTML

### 2. 手动获取特定时间段数据

如果需要获取特定时间段的数据，可以修改脚本中的日期范围，或直接使用 mx-macro-data 技能：

```bash
cd D:\OpenClawData\.openclaw\workspace\skills\mx-macro-data
python scripts/get_data.py --query "美国国债收益率各期限数据，从2023年4月到2026年3月，包括1月期、3月期、6月期、1年期、2年期、3年期、5年期、7年期、10年期、20年期、30年期"
```

然后手动处理数据文件。

## 数据质量检查

脚本会自动进行以下验证：

- ✅ 10年期收益率是否在合理范围（0% - 10%）
- ✅ 是否包含负值（可能混入了TIPS数据）
- ✅ 日期范围是否连续
- ✅ 各期限数据完整性

## 常见问题

### Q: 更新时提示文件被占用？

A: 请确保：
1. 关闭正在查看 CSV 文件的 Excel 或其他程序
2. 关闭浏览器中打开的 HTML 文件
3. 重新运行更新脚本

### Q: 如何查看更新日志？

A: 脚本会输出详细的日志信息，包括：
- 数据获取状态
- 处理的数据量
- 合并后的总行数
- 数据验证结果

### Q: 数据更新频率？

A: 建议每天或每周运行一次更新脚本，以获取最新数据。数据源通常为日度更新。

### Q: 历史数据可以追溯到什么时候？

A: 东方财富数据库通常提供 2020 年以来的完整数据。如需更早数据，需要从其他数据源（如 FRED）获取。

## 技术细节

### 数据源
- **提供商**: 东方财富妙想服务
- **API**: `https://ai-saas.eastmoney.com/proxy/b/mcp/tool/searchMacroData`
- **频率**: 日度
- **更新延迟**: 通常 T+1（次日更新）

## 注意事项

1. **TIPS 数据排除**: 脚本会自动排除通胀保值债券（TIPS）数据，这些数据的实际收益率可能为负值
2. **数据去重**: 合并时会自动去重，保留最新数据
3. **备份建议**: 重要更新前建议备份 `us_treasury_yields_all_terms.csv` 文件
