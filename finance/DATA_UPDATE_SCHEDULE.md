# Tushare 数据更新时间表

## 数据来源确认

### 1. A股日线行情 (daily / index_daily)
- **更新时间**: 交易日每天 **15:00～16:00** 之间入库
- **数据说明**: 未复权行情，停牌期间不提供数据
- **接口**: `index_daily` (指数日线)

### 2. 融资融券交易汇总 (margin)
- **数据来源**: 证券交易所网站直接获取
- **更新时间**: 当日即可更新当日数据
- **说明**: 根据券商申报的数据汇总，由券商保证数据的真实、完整、准确
- **交易所公布时间**: 通常在收盘后 **17:00～18:00** 左右

## 建议定时任务设置

### 方案一：保守方案（推荐）
```bash
# 每天 19:00 执行
0 19 * * * cd /path/to/workspace && python finance/scripts/update_daily_market_data.py
```
**理由**: 
- 指数数据 16:00 前已更新
- 融资融券数据 18:00 前已更新
- 19:00 执行确保所有数据都已入库

### 方案二：实时方案
```bash
# 每天 18:00 执行
0 18 * * * cd /path/to/workspace && python finance/scripts/update_daily_market_data.py
```
**理由**:
- 适合对时效性要求较高的场景
- 融资融券数据可能偶尔延迟

### 方案三：安全方案
```bash
# 每天 20:00 执行
0 20 * * * cd /path/to/workspace && python finance/scripts/update_daily_market_data.py
```
**理由**:
- 确保所有数据都已稳定入库
- 适合数据完整性优先的场景

## 数据更新流程

```
交易日收盘 (15:00)
    ↓
交易所结算 (15:00-17:00)
    ↓
Tushare 指数数据入库 (15:00-16:00) ✓
    ↓
交易所融资融券数据公布 (17:00-18:00)
    ↓
Tushare 融资融券数据入库 (当日) ✓
    ↓
定时任务执行 (建议 19:00)
    ↓
数据更新完成
```

## 注意事项

1. **节假日**: 定时任务会在节假日自动获取最近的数据（脚本已内置7天回溯机制）
2. **数据延迟**: 偶尔会有数据延迟，脚本会自动合并新旧数据
3. **周末**: 周末无新数据，脚本会保持现有数据不变

## 推荐设置

**最佳选择**: 每天 19:00 执行
- 确保所有数据已更新
- 避免数据延迟问题
- 留有足够的数据校验时间
