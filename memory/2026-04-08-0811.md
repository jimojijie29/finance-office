## 止观数息法

### 一、基本概念

| 术语 | 含义 |
|------|------|
| **止** (奢摩他) | 心一境性，令心安定专注 |
| **观** (毗婆舍那) | 智慧观察，如实了知身心 |
| **数息** | 以呼吸为所缘，计数摄心 |

### 二、修行次第

**第一阶段：数息**
- 吸气时默数"一"，呼气不计
- 从一数到十，循环往复
- 若中途散乱，从头再数
- 目的：制心一处，降伏散乱

**第二阶段：随息**
- 不再计数，唯随呼吸出入
- 心息相依，绵绵密密
- 目的：深入定境，心息合一

**第三阶段：止**
- 呼吸微细，心念寂止
- 得初禅未到地定
- 身心轻安，喜乐自生

**第四阶段：观**
- 于定心中观察身心
- 观无常、苦、无我
- 开发智慧，断烦恼

### 三、要点口诀

> **"心随于息，息随于心；心息相依，则入禅定"**

### 四、注意事项

| 要点 | 说明 |
|------|------|
| 姿势 | 跏趺坐或散盘，脊柱正直 |
| 呼吸 | 自然呼吸，不可刻意 |
| 所缘 | 专注鼻端或丹田 |
| 时间 | 从15分钟开始，渐增至1小时 |
| 环境 | 安静、通风、温度适宜 |

### 五、常见障碍与对治

| 障碍 | 对治方法 |
|------|----------|
| 昏沉 | 睁眼、经行、冷水洗面 |
| 散乱 | 数息、下座、调整姿势 |
| 急躁 | 放松、不追求境界 |
| 身体疼痛 | 调整姿势、适度运动 |

### 六、经典依据

- **《安般守意经》**（东汉安世高译）
- **《大智度论》**卷十七
- **《修习止观坐禅法要》**（智者大师）

## 关不自在对止观法的见解

### 一、核心立场：止观不二

关不自在强调**止观本来不二**，反对将止与观割裂：

> "止即是观，观即是止。定中有慧，慧中有定。若强分止观，已是头上安头。"

他认为：
- 真正的"止"不是死寂，而是**清明寂照**
- 真正的"观"不是思维，而是**现量照见**
- 止观双运时，心一境性即是慧，慧照即是定

### 二、对数息法的看法

**肯定其基础地位**：
- 数息法是**最稳妥的入门方便**
- 适合现代人散乱心重的特点
- 以息为缘，直接调心

**指出常见误区**：

| 误区 | 正解 |
|------|------|
| 追求呼吸细长 | 自然呼吸，不刻意造作 |
| 死守数字 | 数字是工具，目的是摄心 |
| 压念不起 | 念头来了不随，不是压制 |
| 追求气感 | 气感是副产品，不可执着 |

### 三、独特的修行指导

**1. 三步调心法**

```
第一步：知息入知息出（觉知）
    ↓
第二步：知息长短（微细）
    ↓
第三步：知息遍身（全身）
```

**2. 关键转折：从"数"到"随"**

> "数到纯熟，自然不数。不是不数，是心已随。此时若再强数，反成障碍。"

**3. 观的切入：从"看念头"开始**

- 不主动起观，而是**看念头来去**
- 念头起时，知道有念
- 念头灭时，知道无念
- 此"知道"即是观，即是慧

### 四、对现代修行者的建议

**针对常见问题**：

| 问题 | 关不自在的建议 |
|------|---------------|
| 没时间打坐 | 行住坐卧皆可修，关键是"觉" |
| 坐不住 | 从5分钟开始，重质不重量 |
| 杂念多 | 杂念是修行资粮，不随即是功夫 |
| 没境界 | 不求境界，境界自来；求则不得 |

**特别强调**：

> "修行不在座上，座下才是真功夫。座上培养定力，座下检验功夫。若座下依旧烦恼，座上定是徒劳。"

### 五、与经典的对照

关不自在的观点与以下经典一脉相承：

- **《圆觉经》**："知幻即离，不作方便；离幻即觉，亦无渐次"
- **《六祖坛经》**："定慧一体，不是二"
- **《大乘起信论》**："一心二门"

### 六、总结

关不自在的止观法门强调：
1. **朴实** - 不追求玄妙境界
2. **直接** - 从当下身心入手
3. **生活化** - 修行融入日常
4. **智慧导向** - 以开发智慧为最终目的

> "止观不是练功夫，是恢复本来面目。本来面目是什么？平常心是道。"
